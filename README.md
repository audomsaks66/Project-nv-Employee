Project: ระบบจองตั๋วรถไฟ
ภาพรวม: 
พัฒนาระบบสำรองตั๋วรถไฟบนเว็บโดยใช้Node.jsและExpressระบบนี้มีฟังก์ชันหลักสำหรับการจัดการการจองของผู้โดยสาร การจัดตารางเวลาของรถไฟ และการตรวจสอบสิทธิ์ผู้ใช้

คุณสมบัติหลัก:

การจัดการการจอง:พัฒนา API เพื่อจัดการการจองตั๋ว รวมถึงข้อมูลผู้โดยสาร การเลือกที่นั่ง และการติดตามสถานะการจอง

การตรวจสอบสิทธิ์ผู้ใช้และความปลอดภัย:ผสานรวมPassport.jsและJWT (JSON Web Token) เพื่อความปลอดภัยของเซสชันผู้ใช้ พร้อมกับการเข้ารหัสรหัสผ่านโดยใช้bcryptเพื่อให้มั่นใจในการปกป้องข้อมูล

การดำเนินการฐานข้อมูล:ใช้Sequelize ORMในการเชื่อมต่อกับSQLiteเพื่อจัดการข้อมูลเชิงสัมพันธ์สำหรับผู้ใช้ การจอง และพนักงานได้อย่างมีประสิทธิภาพ

การตรวจสอบความถูกต้องของข้อมูล:ใช้Joiเพื่อบังคับใช้กฎการตรวจสอบความถูกต้องของข้อมูลอย่างเข้มงวด เพื่อให้มั่นใจในความสมบูรณ์และคุณภาพของข้อมูลที่ระบบประมวลผล

เทคโนโลยีที่ใช้:

แบ็กเอนด์: Node.js, Express

ฐานข้อมูล: SQLite, Sequelize (ORM)

การตรวจสอบสิทธิ์: JWT, Passport.js, bcrypt

เครื่องมือสำหรับการพัฒนา: Joi (การตรวจสอบความถูกต้อง), Postman (การทดสอบ API)
